CopyForceSqlServer -- Copy Salesforce into SQL Server

This directory contains an executable jar (CopyForceSqlServer.jar) that can copy schema/data from a Salesforce instance into SQL Server.

CopyForceSqlServer.jar is an "executable" jar that was built using Java 1.6. To use the jar:

  • java -jar CopyForceSqlServer.jar <command line switches>

For command line switch documentation see the javadoc for com.aslan.sfdc.copyforce.CopyForce.

The jar was built using Java 1.6 and includes all required java classes (e.g. the SOAP classes for chatting with Salesforce). If you need a different version or cannot use Java 1.6 then check out the source and rebuild in Eclipse.
