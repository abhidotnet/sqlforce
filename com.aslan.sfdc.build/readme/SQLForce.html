SQLForce for Jython -- ANSI SQL for Salesforce

SQLForce for Jython is a Jython module that provides ANSI like extensions to SOQL and hides much of the governor limitations imposed by Salesforce.

SQLForce for Jython adds the power of Jython and SOQL level support for the full range of base ANSI SQL commands.

  • SELECT
  • SELECT DISTINCT
  • SELECT .. UNION SELECT
  • UPDATE
  • INSERT
  • DELETE

To get started:

  • Download the current version of Jython and install it.
  • Put sqlforce.jar on your CLASSPATH.
  • Put SQLForce.py somewhere where Jython will pick it up. The Jython directory Lib/site-packages is often a good place
  • Look at the SQLForce for Jython examples.
  • Optionally, read how to create a Salesforce Credentials Registry

The sqlforce.jar jar distributed with SQLForce for Jython is an executable jar that can be ran without Jython using the syntax: 

	java -jar sqlforce.jar
This command line approach to using SQLForce pre-dates integration with Jython and is generally not used. However it is fully supported since SQLForce for Jython is built on it. See SQLForce Without Jython for documentation.

The sqlforce.jar was built using Java 1.6 and includes all required java classes (e.g. the SOAP classes for chatting with Salesforce). If you need a different version or cannot use Java 1.6 then check out the source and rebuild it in Eclipse.
