CopyForceOracle -- Copy Salesforce into Oracle

This directory contains an executable jar (CopyForceOracle.jar) that can copy schema/data from a Salesforce instance into Oracle.

CopyForceOracle.jar is an "executable" jar that was built using Java 1.6. To use the jar:

  • java -jar CopyForceOracle.jar <command line switches>
To get started:
  • Read the command line switch documentation in the javadoc for com.aslan.sfdc.copyforce.CopyForce
  • Read the SQL Server specific command line switch documentation in the javadoc for com.aslan.sfdc.copyforce.CopyForceOracle
  • Optionally, read how to create a Salesforce Credentials Registry

The jar was built using Java 1.6 and includes all required java classes (e.g. the SOAP classes for chatting with Salesforce). If you need a different version or cannot use Java 1.6 then check out the source and rebuild in Eclipse.
