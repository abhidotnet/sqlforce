CopyForceH2 -- Copy Salesforce into H2

This directory contains an executable jar (CopyForceH2.jar) that can copy schema/data from a Salesforce instance into a H2 database.

CopyForceH2.jar is an "executable" jar that was built using Java 1.6. To use the jar:

  • java -jar CopyForceH2.jar <command line switches>

To get started:

  • Read the command line switch documentation in the javadoc for com.aslan.sfdc.copyforce.CopyForce
  • Read the H2 specific command line switch documentation in the javadoc for com.aslan.sfdc.copyforce.CopyForceH2
  • Optionally, read how to create a Salesforce Credentials Registry

The jar was built using Java 1.6 and includes all required java classes (e.g. the SOAP classes for chatting with Salesforce). If you need a different version or cannot use Java 1.6 then check out the source and rebuild in Eclipse.
