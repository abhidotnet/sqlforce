SQLForce for Jython -- Examples

If you are already a Python/Jython programmer, the examples on this page will quickly teach you SQLForce for Jython.

Start with the Connect example to learn how to establish a connection to Salesforce. Once you understand and can run the Connect, all of the other examples will run.

To run any example but Connect, you will need a file named sqlforce.ini in your $HOME (or %USERPFOFILE% on Windows) directory that looks like: 

  <sqlforce>
        <connection 
            name="javaforce"
            type="Production"
            username="gsmithfarmer@gmail.com"
            password="yourpassword"
            token="yoursecuritytokenfromsalesforce"
        />
  </sqlforce>
where the username, password, and token are replaced with your local values.

Script Description
Connect.py Connect to a Salesforce database from SQLForce.
Select.py Select records using SOQL.
SelectDistinct.py Select DISTINCT records using an ANSI extension to SOQL.
SelectUnion.py Combine the results of several SOQL statements using an ANSI SELECT/UNION extension to SOQL.
Insert.py Insert new records using an ANSI INSERT extension to SOQL.
Update.py Update records using an ANSI UPDATE extension to SOQL.
Delete.py Delete records using an ANSI DELETE extension to SOQL.
