Example code illustrating how to perform basic javaforce operations.

javaforce is a java binding the to the Salesforce partner WSDL that hides much the SObject and governer limitations imposed by Salesforce. This package contains example code that illustrates how to connect to salesforce and insert/update/delete/select records.

You are encouraged to look at the Jython module in the SQLForce project as an alternative to coding up Java. The SQLForce module is built on the javaforce library but adds all the power one gets with Python and its modules.
